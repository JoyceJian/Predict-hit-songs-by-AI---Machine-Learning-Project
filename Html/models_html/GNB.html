Gaussian Naive Bayes (GaussianNB)


The next algorithm we tried is Gaussian Naïve Bayes model, where we tried to learn our model from a given set of attributes (in training examples) and then form a hypothesis or a relation to a response variable. Then we use this relation to predict a response, given attributes of a new instance. Using the Bayes’ theorem, it’s possible to build a learner that predicts the probability of the response variable belonging to some class, given a new set of attributes.
